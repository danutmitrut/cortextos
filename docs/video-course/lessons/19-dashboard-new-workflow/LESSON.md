# 19 — Dashboard New Workflow

Duration target: 10-12 minutes

## Learning Outcomes

- Create a new workflow safely.
- Write a good recurring prompt.
- Choose the right owner agent and cadence.

## Script

Start with a simple workflow: "Every weekday morning, review open tasks and send me a short Telegram briefing." Explain the fields: name, agent, schedule, prompt, and safety expectations.

Write the prompt on screen. Include concrete output expectations, required dashboard/task updates, and what to do if blocked. Avoid prompts that say "do everything" or "send external messages automatically."

Create the workflow, then verify it appears on the workflows page and has a reasonable next-fire time.

## Screen Plan

| Time | Show On Screen |
|---|---|
| 0:00 | New workflow page. |
| 2:00 | Fill name/agent/schedule. |
| 4:00 | Draft a clear prompt. |
| 7:00 | Save workflow. |
| 9:00 | Return to workflows list and verify. |

## Learner Checkpoint

Draft one workflow prompt that includes what to inspect, what to produce, and how to report completion.
