# 25 — Template Agents and the Community Catalog

Duration target: 12-15 minutes

## Learning Outcomes

- Understand built-in and community templates.
- Choose a practical first template set.
- Avoid overbuilding the first week.

## Script

Explain templates as starting points for agents with roles, bootstrap files, skills, and setup instructions. Built-in templates cover core roles; community templates provide workflow-specific agents like CRM assistant, project manager, knowledge-base librarian, customer support, automation builder, research, coding, and social media.

Show the community catalog and an agent template folder. Explain that good templates are setup-first: they ask for tuning knobs, tool connections, and workflow preferences instead of assuming the user's exact stack.

Give three starter stacks: solo operator, software team, and AI-native business. Emphasize the CortextOS Concierge as a strong first agent because it helps choose and configure the rest.
