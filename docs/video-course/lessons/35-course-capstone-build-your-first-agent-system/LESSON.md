# 35 — Capstone: Build Your First Agent System

Duration target: 25-35 minutes

## Learning Outcomes

- Combine install, dashboard, template, task, cron, memory, and approval workflows.
- Build a small but real cortextOS operating loop.
- Review the system as an operator.

## Script

Set the capstone goal: build a small first-week agent system. Use a demo org with three agents: Concierge, Knowledge Base Librarian, and Project Manager. If those templates are unavailable in the learner's install, use Orchestrator, Analyst, and a general agent with equivalent roles.

Step one: verify daemon and dashboard. Step two: run setup for the Concierge/starter agent. Step three: ingest one demo doc through the Librarian. Step four: have Project Manager create a three-task implementation plan. Step five: add one recurring workflow, such as weekday morning review. Step six: trigger or simulate one task completion with saved output. Step seven: review dashboard home, tasks, approvals, comms, activity, workflows, and KB.

End with production advice: start with fewer agents, clearer jobs, and stronger review loops. Add more templates only when the workflow is already visible and repeatable.
