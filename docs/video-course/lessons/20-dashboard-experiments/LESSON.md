# 20 — Dashboard Experiments

Duration target: 10-12 minutes

## Learning Outcomes

- Understand autoresearch/theta-wave experiment loops.
- Create hypotheses with measurable outcomes.
- Review experiment results without overclaiming.

## Script

Define an experiment as a targeted change plus a measurement plan. In cortextOS, agents can run research or operational experiments, log the hypothesis, run the change, score the outcome, and decide whether to keep or discard it.

Show the experiments page. Create or inspect a demo experiment: metric, hypothesis, run notes, evaluation, and result. Explain that experiments are not just research essays; they should change behavior if the evidence is good.

Close with examples: improve task-completion quality, reduce stale tasks, improve KB retrieval, or test a content workflow.
