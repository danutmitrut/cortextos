# 16 — Dashboard Strategy Page

Duration target: 8-10 minutes

## Learning Outcomes

- Use strategy/goals as the long-running direction layer.
- Connect goals to tasks and agent behavior.
- Avoid vague goals that cannot be operationalized.

## Script

Explain that strategy is where long-running outcomes live. A good strategy page helps agents know what matters over days or weeks, while tasks handle concrete units of work.

Show goals and strategy notes. Convert a vague goal like "grow the business" into operational goals: publish weekly content, qualify leads, ship onboarding docs, reduce support response time.

Close by showing how an agent should use strategy: query it before planning, then create specific tasks that move it forward.

## Screen Plan

| Time | Show On Screen |
|---|---|
| 0:00 | Strategy page. |
| 2:00 | Example vague goal. |
| 4:00 | Rewrite into measurable operating goals. |
| 6:00 | Create or view task aligned to a goal. |

## Learner Checkpoint

Write one strategic goal and one task that would move it forward this week.
