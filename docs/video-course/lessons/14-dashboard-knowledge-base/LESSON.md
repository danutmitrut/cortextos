# 14 — Dashboard Knowledge Base Page

Duration target: 12-15 minutes

## Learning Outcomes

- Understand the difference between files, memory, and KB.
- Ingest useful knowledge and query it safely.
- Recognize source-quality problems.

## Script

Explain that the knowledge base is associative memory. It lets agents retrieve relevant documents, past research, and shared context. It is not a replacement for structured files or tasks.

Show ingestion, query, and source review. Explain collections/scopes if visible: shared org knowledge, private agent knowledge, and memory collections. Talk about source hygiene: small focused docs beat giant messy dumps.

Close by demonstrating a grounded query and checking whether the answer cites or references the right source.

## Learner Checkpoint

Ingest one demo document and ask a question whose answer is only in that document.
